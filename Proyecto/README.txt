IMPORTANTE
Antes de usar
1. Tener instalado python 3 (última versión)
    https://www.python.org/downloads/
    
2. Comprobar si pip se instaló con python
    Apretar win + R
    Ingresar cmd
    En la pantalla escribir "pip -V"
    Debería aparecer algo parecido a lo siguiente:
        pip 18.0 from c:\users\administrator\appdata\local\programs\python\python37\lib\site-packages\pip (python 3.7)
    
    Si no funciona en el cmd escribir
    curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
    
    Luego
    python get-pip.py
    
    
3. Instalar matplotlib
    Ingresar en el cmd lo siguiente:
    
    pip install matplotlib
    

4. Doble click al archivo llamado "Launcher"
